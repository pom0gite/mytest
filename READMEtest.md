# Base Mini Toolkit

A simple educational project built for the Base ecosystem.

## Overview

The project consists of five smart contracts:

* Token.sol — ERC20 utility token
* NFT.sol — NFT collection contract
* Faucet.sol — token distribution contract
* Voting.sol — governance contract
* Staking.sol — staking and rewards contract

## Goals

* Learn Base smart contract deployment
* Explore governance mechanics
* Test token distribution models
* Build a simple staking system
